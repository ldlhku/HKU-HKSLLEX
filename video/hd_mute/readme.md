# About this folder (HD mute videos)

This folder currently contains only this README and no video files. Full, high-resolution (HD) videos are large and were not included in the GitHub repository due to storage constraints.

Where to find the videos:

- Reduced-size versions (suitable for the sign browser and most use cases) are available in `video/small` and `video/webp`.
- To view the dataset in your browser, use `sign_browser.html` (local) or `sign_browser_github.html` (hosted on GitHub Pages) â€” these will use the reduced-size videos if present.

If you require access to the original full-resolution HD videos for research, please contact the repository maintainers; they can provide access instructions or direct links to the full videos.
